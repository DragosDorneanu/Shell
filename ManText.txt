Man Page
Here you will find information about all of the available functions.
For information about how to use a certein function type "man functionName".
Press 'Q'(quit) to exit the Man Page.


Available functions :
	
    * exit : Close console(shell).
    * clear : Type "clear" in shell console to clear all entered commands or outputed information.
    * cd : Change directory.
    * cat : Display a text file's content on screen.
    * find : Find if a file with given name as parameter exists or not in the current hierarchy of files.
    * list : Display an array of strings denoting all files in the current directory or in the directory given as parameter.
    * open : Open a file(.txt, .png, .flac, .exe etc.) or a folder.
    * mkdir : create a new directory in current directory haveing the name given as parameter
    * remove : remove from the hard drive the specified file if it exists
    * create : create a new file having the specified name and opens it for edit
