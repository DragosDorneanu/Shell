Remove Man Page
Here you will find information about "remove" function.

Basic information :
	* delete an existing file from the current hierarchy of files

Available options:
	
	* -r [directory] : recursive delete all files contained by the directory given as parameter and the direcotry
	* [file1] [file2] ... [fileN] : delete all files given as parameter ( it can be an empty direcotry as well )
